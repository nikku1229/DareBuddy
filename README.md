# 🎯 Dare Game App

A fun and interactive multiplayer **Dare Game** built using the **MERN Stack**.
Play with friends, roll dice, and complete exciting dares based on categories and game types.

---

## 🚀 Live Demo

🔗 **Frontend:** https://dare-buddy.vercel.app/

---

## 📌 Features

- 🎲 Dice rolling system (1–100)
- 👥 Multiplayer support
- 🎯 Dynamic dare generation from database
- 🗂️ Category-based dare selection
- 🎮 Multiple game types (customizable)
- 📱 Fully responsive UI (mobile, tablet, desktop)
- ⚡ Smooth animations and transitions
- 🔒 Rate limiting & backend security

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/nikku1229/DareBuddy.git
cd darebuddy
```

---

### 2️⃣ Setup Backend

```bash
cd server
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
CLIENT_URL=http://localhost:5173
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```
